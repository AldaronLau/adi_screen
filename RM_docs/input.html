Input

use adi_screen::Input;

Input represents user input.

Redraw

Since the user is not inputing anything, the screen needs to be redrawn.

Resize

The window has been resized, sprites dependant on aspect ratio need to be resized.

Back

The back key, escape key, or start button has been pressed, or the user has clicked the close (X) button on the window. It is the programmer's responsibility to break out of the main loop at this point, or prevent the program from closing.

Resume

The window has moved into focus.

Pause

The window has moved out of focus.

EnterWindow

Cursor moved into window

LeaveWindow

Cursor moved out of window

KeyDown(adi_screen::Key)

A key is pressed down.

KeyRepeat(adi_screen::Key)

A printable key is held down long enough to repeat.

KeyUp(adi_screen::Key)

A key is released.

Cursor(f32,f32)

The mouse moves.

LeftDown(f32,f32)

The left (primary) mouse button is clicked.

MiddleDown(f32,f32)

The middle mouse button is clicked.

RightDown(f32,f32)

The right mouse button is clicked.

LeftUp(f32,f32)

The left mouse button is released.

MiddleUp(f32,f32)

The middle mouse button is released.

RightUp(f32,f32)

The right mouse button is release.

ScrollUp(f32,f32)

The scroll wheel is moved up.

ScrollDown(f32,f32)

The scroll wheel is moved down.

ScrollLeft(f32,f32)

The scroll wheel is moved left.

ScrollRight(f32,f32)

The scroll wheel is moved right.

Key

use adi_screen::Key;

Key represents a key on a computer keyboard.

Char(char)

a-z & 0-9 -=[];',./\`-*+ ( Printable keys ).

Backspace

The Backspace key.

Delete

The Delete key.

Ctrl(bool)

True if right Ctrl key, False if left Ctrl key.

Shift(bool)

True if right Shift key, False if left Shift key.

Alt(bool)

True if right Alt key, False if left Alt key.

Compose

The CapsLock or Compose key.

NumLock

The NumLock key.

Home

The Home key.

End

The End key.

PageUp

The PageUp key.

PageDown

The PageDown key.

Up

The Up key.

Down

The Down key.

Left

The Left key.

Right

The Right key.

Insert

The Insert key.
