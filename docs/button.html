Button

use adi_screen::Button;

Button represents a clickable gui button.

fn create(window: &mut Window, pos: (f32,f32)) -> Button

Create a new button at position pos.

fn update(&mut self, window: &mut Window, input: Input) -> bool

Update the state of the button. Pass input from Window::update() to input. Returns true if the button has been pressed, and false otherwise.
